To hide sections of a Foswiki topic, use HTML comments. 

Example comment within a sentence:

=Visible &lt;!--- and hidden ---&gt; text=

Example spanning multiple lines:
<verbatim>
<!--
   * Useful to hide settings:
      * Set EXAMPLE = This is hidden
-->
</verbatim>
